# .claude

Claude Code configuration and slash commands for the SwiftUIML project.

## Commands

- `/add-view` — Add a new view type following project patterns
- `/add-modifier` — Add a new modifier following project patterns
- `/add-snapshot` — Create a snapshot test for a renderer
- `/add-standardized-test` — Create a standardized test with Swift and TypeScript implementations
- `/test-json` — Test JSON serialization/deserialization
- `/code-review` — Review staged changes or branch against main
- `/audit` — Comprehensive project audit
- `/audit-docs` — Documentation audit
- `/audit-tools` — Verify commands are accurate against the codebase
